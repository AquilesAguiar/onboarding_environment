# Onboarding
## Instruções
- Faça um fork desse repositório.
- Crie um branch a partir do master para cada desafio.
- Crie um PR do branch do desafio para o master quando finalizar.
## Instalação
```bash
docker-compose build
```

## Ambientes

- [Rails](rails)
- [Phoenix](phoenix)

## Ferramentas

### Mongo

Iniciar o mongo
```bash
docker-compose up -d mongo
```
